# Sanity Studio Template

A clean and ready-to-use **Sanity Studio template** with support for categories, featured projects, services, settings, SEO, and fully **localized content** for translating strings, blocks, and text. Ideal as a base for modern content-driven projects.  

---

## 🚀 Features

- **Categories & Featured Projects** – Organize your content and highlight key projects.  
- **Services** – Manage service offerings easily.  
- **Settings & SEO** – Centralized configuration for site settings and SEO metadata.  
- **Localized Content** – Translate strings, blocks, and rich text across multiple languages (e.g., English, Spanish, Catalan).  
- **Clean Folder Structure** – Ready to extend with custom schemas and desk structure.  
- **TypeScript & Modern Setup** – Safer code and smooth developer experience.  

---

## 💻 Getting Started

1. **Install dependencies**

	•	npm install
	•	or
	•	yarn install

2. **Run the Studio locally**

	•	npm run dev
	•	or
	•	yarn dev

Open http://localhost:3333￼ in your browser to access the Studio.

3. **Deploy Studio**

Check out Sanity Deployment Docs￼ for deploying to Vercel, Netlify, or Sanity’s managed hosting.

⸻

## 📂 Project Structure

/schemas       # All content schemas: categories, projects, services, settings, SEO
/documents     # The schemas that appear in the sidebar.
/objects       # Personalized schemas for translation, SEO and other.
